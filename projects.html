QiT

Python Flask JavaScript HTML CSS Jinja PostgreSQL

QiT is a fully functional full stack flask web app that is intended for students to communicate and reach out for help on school related subjects. I was heavily inspired to make this app by Reddit, Quizlet, and Stackoverflow. During this project, I learned a lot about the flask python framework and other useful skills such as SQL and Jinja. Jinja made viewing different user pages and handling conditionals in HTML super easy. The biggest struggle I had during this project was managing relationships between columns inside my database. I poured a majority of the hours learning and relearning how database relationships and columns worked. Eventually after grasping the subject a little better, I was able to add other cool functions such as commenting and liking posts. I still have a lot to learn when it comes to making full stack applications, however this was definitely a huge milestone for me.

Sorting Visualizer

JavaScript HTML CSS

The sorting visualizer was one of the most challenging projects I had when first learning front-end web development. DOM manipulation was a hard concept for me to really get used to. This project was really fun and looking back at it makes me feel very proud about how much I had accomplished.
